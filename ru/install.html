Установка Aivika

Инструкции ниже приведены для случая, если вы используете Stack.

Использование только библиотек с открытыми исходниками

При использовании только библиотек Aivika с открытыми исходниками, вы можете добавить следующие зависимости в секцию extra-deps вашего файла stack.yaml:

- aivika-5.5
- aivika-transformers-5.5
- aivika-distributed-1.0
- aivika-experiment-5.2
- aivika-experiment-chart-5.1
- aivika-experiment-diagrams-5.1
- aivika-experiment-cairo-5.1
- aivika-realtime-0.3
- aivika-branches-0.3
- aivika-lattice-0.6
- aivika-gpss-0.4

Теперь эти библиотеки доступны в вашем проекте Stack.

Использование Aivika Extension Pack

Aivika Extension Pack – это множество дополнительных библиотек Aivika, правила использования которых покрываются следующей лицензией.

Чтобы включить эти библиотеки в ваш проект Stack, вам следует добавить следующие элементы в секцию packages вашего файла stack.yaml, как показано ниже:

- location:
   https://github.com/dsorokin/aivika-experiment-entity/archive/v0.4.tar.gz
  extra-dep: true
- location:
   https://github.com/dsorokin/aivika-experiment-entity-HDBC/archive/v0.4.tar.gz
  extra-dep: true
- location:
   https://github.com/dsorokin/aivika-experiment-provider/archive/v0.4.tar.gz
  extra-dep: true
- location:
   https://github.com/dsorokin/aivika-experiment-provider-distributed/archive/v0.2.tar.gz
  extra-dep: true
- location:
   https://github.com/dsorokin/aivika-experiment-report/archive/v0.4.tar.gz
  extra-dep: true
- '.'

Затем вам следует определить соответствующие зависимости в секции extra-deps того же файла:

- aivika-experiment-entity-0.4
- aivika-experiment-entity-HDBC-0.4
- aivika-experiment-provider-0.4
- aivika-experiment-provider-distributed-0.2
- aivika-experiment-report-0.4

Теперь вы можете использовать Aivika Extension Pack в вашем проекте Stack.

Документация по API

После установки вы можете сгенерировать документацию по API следующим образом:

$ stack haddock
