Документация Aivika

Aivika разделена на две части. Главные библиотеки лицензированы по BSD3. В большинстве случаев будет достаточно использовать только их.

В тоже время, если вам нужно запустить численный эксперимент для распределенной имитации, к примеру, то вполне вероятно, что вам понадобится Aivika Extension Pack, использование которого регламентируется следующей лицензией.

Документация PDF

Основные концепции Aivika описаны в следующем документе PDF (черновик):

  • Aivika: Computation-based Modeling and Simulation in Haskell

Документация по API

Лицензированные по открытой лицензии BSD3 библиотеки Aivika расположены на Hackage DB, где доступна документация по их API:

  • http://hackage.haskell.org/package/aivika

  • http://hackage.haskell.org/package/aivika-transformers

  • http://hackage.haskell.org/package/aivika-distributed

  • http://hackage.haskell.org/package/aivika-experiment

  • http://hackage.haskell.org/package/aivika-experiment-chart

  • http://hackage.haskell.org/package/aivika-experiment-diagrams

  • http://hackage.haskell.org/package/aivika-experiment-cairo

  • http://hackage.haskell.org/package/aivika-realtime

  • http://hackage.haskell.org/package/aivika-branches

  • http://hackage.haskell.org/package/aivika-lattice

  • http://hackage.haskell.org/package/aivika-gpss

Что касается Aivika Extension Pack, то вы можете сами сгенерировать документацию по API после установки Aivika Extension Pack, как описано в инструкции по установке.

Примеры

Также многие пакеты содержат множество примеров и тестов. Вы можете использовать их, чтобы научиться применять Aivika.
