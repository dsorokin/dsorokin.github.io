Aivika for .NET

Aivika for .NET is a simplified .NET port of the Aivika simulation framework. It implements only a small part of Aivika. The port is written in the F# programming language.

Aivika for .NET is available under a dual-license model: GNU GPL License v3 (GPLv3) and Commercial. Please see the license.

Please contact for detail of licensing using the information provided by the Contact page.

The sources can be found on GitHub:

  • https://github.com/dsorokin/aivika-fsharp-ce

There is the PDF documentation that describes the method, main API and provides with some examples with graphical illustrations:

  • Aivika 3 User Guide: Version for .NET Framework and Mono
